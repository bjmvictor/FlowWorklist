As informacoes abaixo devem ser configuradas no console/modalidade que fara o envio da imagem para esta impressora virtual.

Nome da Application Entity da impressora virtual. Exemplo: VPRINTSCP.

Porta TCP onde a modalidade vai conectar para DICOM Print. Exemplo: 4100.

Nome do perfil no arquivo de configuracao gerado para o DCMTK.

Diretorio que contem dcmprscp.exe e dcm2img.exe. Exemplo: C:\dcmtk\bin.

Endereco usado pelo perfil LOCALPRINTER do DCMTK. Exemplo: 127.0.0.1.

Worker de Conversao e Impressao

Pasta monitorada para arquivos HG_*.dcm.

Pasta de spool usada pelo receptor DICOM Print.

Pasta usada para arquivos temporarios convertidos para impressao.

Executavel usado para enviar PDF para impressora Windows. Exemplo: C:\Program Files\SumatraPDF\SumatraPDF.exe.

Nome da impressora exatamente como aparece no Windows.

Tamanho de pagina do PDF gerado para impressao.

Parametros usados no -print-settings. Exemplo: fit, shrink, noscale.

Ao limpar SP relacionado, usa diferenca de tempo em segundos. Exemplo: 120.

Tempo entre verificacoes da pasta de entrada. Exemplo: 1.0.
